# PhotonBox

**The TV operating system you actually own.**

PhotonBox is a next-generation, open-source OS environment designed to take back the living room. Built to be lightweight, hardware-agnostic, and entirely modular, it replaces ad-ridden consumer TV interfaces with a clean, user-controlled ecosystem.

> *"Fighting to get out of the box."*

---

## ⚡ Core Vision & Killer Features

*   **Hardware Agnostic (x86 & ARM):** Whether you're running a repurposed x86 thin client or a standard ARM-based TV box, PhotonBox adapts. 
*   **HomeLab First:** Native, out-of-the-box integration with the self-hosted ecosystem. Your media, your servers (Jellyfin, Plex, local NAS), front and center.
*   **Gaming Ready:** Built-in support for local streaming networks (like SteamLink on supported x86 hardware) and game controller management.
*   **Smart Environment:** Deep hooks into Smart Home ecosystems. The TV isn't just a screen; it's a node in your network (ready for Home Assistant, ambient lighting sync, and custom dashboards).
*   **Zero Telemetry:** No ads, no tracking, no forced cloud accounts.
